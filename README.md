# Project: Web Utility Suite

本リポジトリは、AIによる保守・開発を前提としたWebツール群です。

## 1. 保守方針と設計思想
- **Type A (ポータブル):** HTML1枚完結。ビルド不要。
- **Type B (コンポーネント):** TS/React/Vite。ビルド必須。
- **原則:** AI-Friendlyであること。外部依存は最小限に。

## 2. 外部サービス依存
- **Firebase:** リアルタイム同期
- **Streamlit:** 高負荷計算
- **GitHub Pages:** ホスティング

## 3. 開発・修正フロー
1. 対象ファイルを確認（Type A or B）。
2. AIに対し「設計意図」を伝え、修正を依頼。
3. `history.json` にログを追記。
4. `README.md` の「変更履歴」を追記。
